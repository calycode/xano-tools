# oc serve
>[!NOTE|label:Description]
> #### Serve the OpenCode AI server locally.

```term
$ xano oc serve [options]
```
### Options

#### --port <port>
**Description:** Port to run the OpenCode server on (default: 4096)
#### -d, --detach
**Description:** Run the server in the background (detached mode)

### oc serve --help
